SYSTEM.OS — Solo Leveling–Inspired Life Operating System

SYSTEM.OS is a Solo Leveling–inspired self-improvement web application that turns real-life discipline, learning, and consistency into an RPG-style progression system.

You are the player.
Your daily actions are quests.
Progress is earned through execution, not motivation.

🧠 Core Philosophy

SYSTEM.OS is built on one principle:

Real growth must be earned, measured, and sustained.

There are no fake progress bars, instant rewards, or passive tracking.
Every stat increase, level-up, and reward is tied to completed actions and proven consistency.

⚙️ Key Features

🎯 Quest-Based Progression
Daily, optional, and skill-bound quests represent real-world actions.

🧬 Stat System with Progress Bars
Stats increase only when progress bars are filled through relevant actions.

🔁 Skill Protocols
Each skill (Fitness, Learning, Development, etc.) progresses independently through execution history and streaks.

🧠 Mentor AI (Gemini API)
Interprets user reports and generates skill-specific guidance and challenges.

🔥 Boss Fights & Milestones
Boss encounters every few levels test discipline, consistency, and growth.

🛡️ Penalty System
Skipped quests generate penalty tasks to enforce accountability.

🧿 Vision & Anti-Vision System
Tracks who you want to become—and who you refuse to become—to guide decisions and motivation.

💾 Daily Rollback Safety
System can reset to the start of the day to recover from bugs without losing long-term progress.

🧩 Design Goals

Execution > Motivation

Proof over time tracking

Progress without gimmicks

Stable systems over flashy UI

RPG immersion grounded in real behavior

🛠 Tech Stack

Frontend: React + TypeScript

State: LocalStorage-based persistence

AI: Gemini API (BYOK supported)

Design: System-UI / dark sci-fi aesthetic
